# 构建
- 工程目录下执行sh build.sh会构建生成libnebuladb.so
- 测试目录下执行sh build.sh会生成对应的可执行文件
# 关键约定
- 我会在doc一个一个模块的写实现原理，然后你帮我实现，如果实现的模块代码中依赖未开发的模块的接口，你需要在未开发的模块中写一个函数，直接返回。
- 类名使用驼峰规则，其它使用C++ 谷歌风格
